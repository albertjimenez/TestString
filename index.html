Bienvenido a Gestión Hospital Android

Esta página recogerá la distinta documentación e ilustración acerca del proyecto y el estado en el que se encuentra. Actualmente, se conecta a Firebase con autenticación mediante cuenta de Google para asegurar unicidad en la base de datos. Cada usuario representa un medico con una lista de pacientes a su cargo pero con acceso de lectura para el resto de la base de datos. De este modo las operaciones que se pueden hacer en este momento son:

  • Crear nuevos usuarios con datos propios
  • Crear nuevos usuarios con datos aleatorios
  • Consultar usuarios en la base
  • Deshacer la creación de usuarios
  • Personalizar brevemente la foto de perfil de usuario.
